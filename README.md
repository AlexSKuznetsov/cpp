# Basic C++ GoogleTests setup with VS Code

https://google.github.io/googletest/quickstart-cmake.html

### Manual build and run tests

1. Open terminal

```bash
 cmake -S . -B build
```

```bash
cmake --build build
```

```bash
cd build && ctest
```
